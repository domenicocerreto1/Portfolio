# Race Engineering Tools (Demo)

A modular platform to support trackside performance workflow: session/run organization, fast setup comparison, tyre/fuel estimation, driver debrief/telemetry review, and race strategy simulation.

## Live demo
[https://lnkd.in/dDxDgdni](https://race-engineer-app-1.streamlit.app/)

## Presentation (PDF)
- Download: ./Race_Engineering_Tools_Presentation.pdf

## Key features
- Run Sheet for session/run management and export
- Setup Comparator with derived KPIs and warnings
- Tyre pressure (k-factor calibration) & fuel estimation
- Driver Debrief & telemetry visualization
- Strategy builder + optimizer + strategy comparison plots
